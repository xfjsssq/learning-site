# 第四章：极限与连续

## 什么是极限？

极限是微积分的基础概念。它描述了一个变量在某个变化过程中无限趋近于某个确定值的现象。

### 数列极限

设数列 {aₙ}，如果存在常数 A，对于任意给定的正数 ε（无论它多么小），总存在正整数 N，使得当 n > N 时，|aₙ - A| < ε，则称常数 A 是数列 {aₙ} 的极限。

### 函数极限

当 x → x₀ 时，f(x) 的极限定义为：对于任意 ε > 0，存在 δ > 0，当 0 < |x - x₀| < δ 时，|f(x) - A| < ε。

## 极限的基本性质

| 性质 | 描述 |
|------|------|
| 唯一性 | 如果极限存在，则极限值唯一 |
| 有界性 | 收敛数列必有界 |
| 保号性 | 若极限 > 0，则存在邻域使函数值 > 0 |
| 夹逼定理 | f(x) ≤ g(x) ≤ h(x)，lim f = lim h = A，则 lim g = A |

## 两个重要极限

1. **第一个重要极限**：lim(x→0) sin(x)/x = 1
2. **第二个重要极限**：lim(x→∞) (1 + 1/x)^x = e

## 例题

求极限：lim(x→0) (1 - cos x) / x²

**解**：使用等价无穷小替换，1 - cos x ~ x²/2

lim(x→0) (x²/2) / x² = 1/2
