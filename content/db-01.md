# 第一章：SQL 基础与进阶

SQL 是数据分析、后端开发的核心技能。

## 基础查询

```sql
SELECT col1, col2
FROM table
WHERE condition
GROUP BY col1
HAVING count(*) > 1
ORDER BY col1 DESC
LIMIT 10;
```

## 连接查询

| 连接类型 | 说明 |
|----------|------|
| INNER JOIN | 两表都匹配的行 |
| LEFT JOIN | 左表全保留，右表匹配则显示 |
| RIGHT JOIN | 右表全保留 |
| FULL JOIN | 两表全保留（MySQL 不支持，用 UNION 模拟） |

## 窗口函数

```sql
-- 排名
ROW_NUMBER() OVER (PARTITION BY dept ORDER BY salary DESC) AS rn
RANK() / DENSE_RANK()

-- 偏移
LAG(salary) OVER (PARTITION BY dept ORDER BY hire_date) AS prev_salary
LEAD(salary) OVER (...)
```

## 索引优化

- **覆盖索引**：查询列全在索引中，避免回表
- **最左前缀**：联合索引 (a,b,c) 支持 a、a,b、a,b,c
- **避免**：`WHERE col % 2 = 0`、函数调用、隐式类型转换

## 事务与隔离级别

| 级别 | 脏读 | 不可重复读 | 幻读 |
|------|------|------------|------|
| READ UNCOMMITTED | ✓ | ✓ | ✓ |
| READ COMMITTED | ✗ | ✓ | ✓ |
| REPEATABLE READ | ✗ | ✗ | ✓ (MySQL 默认，用 Next-Key Lock 解决) |
| SERIALIZABLE | ✗ | ✗ | ✗ |

> 你的学习记录按用户隔离存储，数据安全无顾虑。
