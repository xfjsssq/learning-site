# 第五章：导数与微分

## 导数的定义

导数是描述函数在某一点处变化率的数学工具。

### 形式化定义

设函数 y = f(x) 在点 x₀ 的某邻域内有定义，如果极限

lim(Δx→0) [f(x₀ + Δx) - f(x₀)] / Δx

存在，则称此极限为 f(x) 在点 x₀ 处的导数，记作 f'(x₀) 或 dy/dx |_{x=x₀}。

### 导数的几何意义

导数 f'(x₀) 就是曲线 y = f(x) 在点 (x₀, f(x₀)) 处切线的斜率。

## 基本导数公式

| 函数 | 导数 |
|------|------|
| f(x) = c | f'(x) = 0 |
| f(x) = xⁿ | f'(x) = nxⁿ⁻¹ |
| f(x) = sin x | f'(x) = cos x |
| f(x) = cos x | f'(x) = -sin x |
| f(x) = eˣ | f'(x) = eˣ |
| f(x) = ln x | f'(x) = 1/x |
| f(x) = aˣ | f'(x) = aˣ ln a |

## 求导法则

- **和差法则**：(f ± g)' = f' ± g'
- **乘积法则**：(fg)' = f'g + fg'
- **商法则**：(f/g)' = (f'g - fg')/g²
- **链式法则**：[f(g(x))]' = f'(g(x)) · g'(x)

## 导数的应用

1. **求切线方程**：y - f(x₀) = f'(x₀)(x - x₀)
2. **单调性判别**：f'(x) > 0 → 递增，f'(x) < 0 → 递减
3. **极值问题**：f'(x₀) = 0 且 f''(x₀) ≠ 0 → 极值点
