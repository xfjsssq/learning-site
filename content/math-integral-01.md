# 第六章：积分学入门

## 不定积分

积分是导数的逆运算。如果 F'(x) = f(x)，则 F(x) 称为 f(x) 的一个原函数。

### 基本积分公式

| 函数 | 积分 |
|------|------|
| ∫ xⁿ dx | xⁿ⁺¹/(n+1) + C (n ≠ -1) |
| ∫ 1/x dx | ln|x| + C |
| ∫ eˣ dx | eˣ + C |
| ∫ aˣ dx | aˣ/ln a + C |
| ∫ sin x dx | -cos x + C |
| ∫ cos x dx | sin x + C |

## 定积分

定积分表示曲线下的面积：

∫[a,b] f(x) dx = F(b) - F(a)

其中 F(x) 是 f(x) 的原函数。

### 微积分基本定理

如果 f(x) 在 [a, b] 上连续，F(x) 是 f(x) 的一个原函数，则：

∫[a,b] f(x) dx = F(b) - F(a)

## 换元积分法

第一换元法（凑微分）：
∫ f(g(x))g'(x) dx = ∫ f(u) du （令 u = g(x)）

第二换元法（变量代换）：
令 x = φ(t)，则 ∫ f(x) dx = ∫ f(φ(t))φ'(t) dt

## 分部积分法

∫ u dv = uv - ∫ v du

口诀：选 u 遵循"反对幂指三"（反三角、对数、幂函数、指数、三角函数）
