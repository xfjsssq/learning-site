# 第八章：无穷级数

## 常数项级数

### 基本概念

级数 ∑ₙ₌₁^∞ aₙ 收敛 ⇔ 部分和数列 {Sₙ} 有极限，极限值称为级数的和。

### 正项级数的判别法

| 判别法 | 条件 | 结论 |
|--------|------|------|
| 比较判别法 | 0 ≤ aₙ ≤ bₙ，∑bₙ 收敛 | ∑aₙ 收敛 |
| 比值判别法 | lim aₙ₊₁/aₙ = ρ | ρ < 1 收敛，ρ > 1 发散 |
| 根值判别法 | lim ⁿ√aₙ = ρ | ρ < 1 收敛，ρ > 1 发散 |
| 积分判别法 | f(x) 单调递减 | ∑aₙ 与 ∫f(x)dx 同敛散 |

### 任意项级数

- **交错级数**：莱布尼茨判别法 — 若 |aₙ| 单调递减且趋于 0，则 ∑(-1)ⁿ⁻¹aₙ 收敛
- **绝对收敛**：∑|aₙ| 收敛 ⇒ ∑aₙ 收敛
- **条件收敛**：∑aₙ 收敛但 ∑|aₙ| 发散

## 幂级数

### 形式

∑ₙ₌₀^∞ aₙ(x - x₀)ⁿ

### 收敛半径

R = lim |aₙ/aₙ₊₁|（比值法）
或 R = 1/lim ⁿ√|aₙ|（根值法）

### 常见展开式

- eˣ = ∑ₙ₌₀^∞ xⁿ/n! ，x ∈ (-∞, +∞)
- sin x = ∑ₙ₌₀^∞ (-1)ⁿx²ⁿ⁺¹/(2n+1)! ，x ∈ (-∞, +∞)
- cos x = ∑ₙ₌₀^∞ (-1)ⁿx²ⁿ/(2n)! ，x ∈ (-∞, +∞)
- 1/(1-x) = ∑ₙ₌₀^∞ xⁿ ，x ∈ (-1, 1)

## 傅里叶级数

任何周期为 2π 的函数可以展开为：

f(x) = a₀/2 + ∑(aₙcos(nx) + bₙsin(nx))

其中：
- aₙ = (1/π) ∫[-π,π] f(x)cos(nx) dx
- bₙ = (1/π) ∫[-π,π] f(x)sin(nx) dx
