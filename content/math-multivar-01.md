# 第七章：多元函数微分

## 多元函数的极限与连续

多元函数研究的是多个自变量的函数关系，如 z = f(x, y)。

### 二元函数的极限

lim((x,y)→(x₀,y₀)) f(x, y) = A 的含义是：无论 (x, y) 以何种方式趋近于 (x₀, y₀)，函数值都无限接近 A。

### 连续性

如果 lim((x,y)→(x₀,y₀)) f(x, y) = f(x₀, y₀)，则函数在该点连续。

## 偏导数

偏导数是多元函数对其中一个自变量的导数，其他变量视为常数。

### 定义

∂z/∂x = lim(Δx→0) [f(x₀ + Δx, y₀) - f(x₀, y₀)] / Δx

### 几何意义

偏导数 ∂z/∂x 表示曲面 z = f(x, y) 与平面 y = y₀ 交线在点 (x₀, y₀) 处切线对 x 轴的斜率。

## 全微分

全微分描述了多元函数在所有自变量都变化时的总变化量：

dz = (∂z/∂x)dx + (∂z/∂y)dy

### 可微与连续的关系

可微 → 连续，连续 ↛ 可微
可微 → 偏导存在，偏导存在 ↛ 可微

## 多元函数求极值

### 无条件极值

1. 求驻点：∂z/∂x = 0，∂z/∂y = 0
2. 用二阶偏导数判别法：记 A = f''ₓₓ, B = f''ₓᵧ, C = f''ᵧᵧ
   - AC - B² > 0 且 A > 0 → 极小值
   - AC - B² > 0 且 A < 0 → 极大值
   - AC - B² < 0 → 不是极值
